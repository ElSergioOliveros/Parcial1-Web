# Parcial 1 - Sergio Oliveros

## Instalación y ejecución

* Clonar repositorio:

```bash
git clone https://github.com/ElSergioOliveros/Parcial1-Web.git
```

* Instalar dependencias:

```bash
cd parcial1
npm install
```

* Ejecutar:

```bash
npm start
```
